Como Jogar

Nim é um jogo de estratégia matemática!

Existem três pilhas distintas com 3, 5 e 7 itens cada.

Em cada turno, você deve remover pelo menos um item de uma pilha. Sua vez é indicada pelo botão destacado com seu nome.

Você pode remover quantos itens quiser dessa única pilha. Depois de remover quantos itens desejar, pressione seu nome para trocar de jogador.

O jogador que restar com o último item do tabuleiro perde!

Jogando contra o computador...

O jogador do computador implementa um algoritmo para reduzir a soma digital binária (também chamada de soma nim) dos itens a 0 em cada turno, o que garante uma vitória, a menos que você jogue perfeitamente!
